🔥 Firebase афзалликлари:
✅ Ҳақиқий реал-тайм (1 сония)

Firebase Realtime Database URL
Маълумотлар сақланадиган йўл
Ёки мавжуд маълумотларни Firebase га юкланг:
📋 Firebase созлаш:
1. Firebase Console га киринг
2. Янги лойиҳа яратинг
3. Realtime Database → Create
4. Rules: ".read": true, ".write": true
5. Database URL ни нусхаланг
Google Sheets ҳавола нусхасини қўйинг
Бўш қолдирсангиз биринчи варақ олинади
⚠️ Тўғридан-тўғри уланиш ишламади
Браузер хавфсизлик чекловлари сабабли. Қуйидаги усулни ишлатинг:
Файл юкланганидан сўнг, Excel / CSV табидан юкланг
✅ Усул 1: Publish to Web
1. File → Share → Publish to web
2. Publish босинг
3. Чиққан URL ни қўйинг (2PACX-... бошланади)
⚠️ Google 5 дақиқа кэшлайди - янгиланиш кечикиши бўлади
⚡ Усул 2: Оддий URL (тезроқ)
1. Google Sheets да Share → Anyone with link
2. Браузер манзилидан URL ни нусхаланг
3. Ўша URL ни шу ерга қўйинг
